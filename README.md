## Встановлення залежностей
1. Клонуйте репозиторій:
   `git clone <LINK>`
3. Перейдіть в директорію із проектом.
4. У папці з проектом (де файл з розширенням .csproj) відкрийте консоль та введіть команду:
   `dotnet restore`

## Запуск проекту
1. Перейдіть в директорію із проектом.
2. Виконайте команду в терміналі:
   `dotnet run`

## Запуск перевірки лінтером
Лінтер StyleCop автоматично запускається під час компіляції або запуску проекту. 
Для компіляції відкрийте термінал в папці проекту і введіть: 
`dotnet build`

## Структури даних
Оскільки матимемо справу з графом, то його буде доцільно представити як асоціативний словник, а конкретно у випадко C# - Dictionary. Тоді ключем буде пакет, а значенням - список пакетів, від яких залежить цей пакет (це буде представлено динамічним масивом - List).
Для встановлення порядку встановлення використовуватиметься алгоритм топологічного сортування, який потребуватиме використання стеку (Stack) та колекції HashSet.
