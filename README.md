# Dependency Resolver

## Встановлення залежностей
1. Клонуйте репозиторій:
   `git clone <LINK>`
3. Перейдіть в директорію із проектом.
4. У папці з проектом (де файл з розширенням .csproj) відкрийте консоль та введіть команду:
   `dotnet restore`

## Запуск проекту
1. Перейдіть в директорію із проектом.
2. Виконайте команду:
   `dotnet run`

## Запуск перевірки лінтером
Лінтер StyleCop автоматично запускається під час компіляції або запуску проекту. 
Для компіляції відкрийте термінал в папці проекту і введіть: 
`dotnet build`
